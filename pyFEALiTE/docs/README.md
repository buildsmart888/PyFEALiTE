# PyFEALiTE Documentation

## GitHub Pages Setup

This directory contains the documentation for PyFEALiTE.

### Features Documented:
- Professional 2×3 Layout Visualization System
- Auto-Scale Optimization for Force Diagrams
- Global & Element-wise Max/Min Values
- Material & Section Integration
- LoadDirection.Global Compliance
- Complete API Reference

### Building Documentation:

```bash
pip install sphinx sphinx-rtd-theme
cd docs
make html
```

The documentation will be automatically built and deployed via GitHub Actions.
